lancia i seguenti comandi nel progetto :
1) npm init 
2) npm i -o vite
2.1) aggiungi nel package.json , sotto l'elemento scripts , esempio : 
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev": "vite",
    "build": "build vite"
  },
3) npm run dev
